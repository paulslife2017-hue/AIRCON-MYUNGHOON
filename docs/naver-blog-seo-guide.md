## 7. 네이버 블로그 SEO 핵심 원칙 (2026 기준)

### 7-1. 네이버 검색 노출 알고리즘 포인트

```
1. 제목 키워드 정확성 (30%)
   → "[지역] [서비스] 출장 후기" 형태가 최고

2. 본문 키워드 밀도 (20%)
   → 1,500자 이상, 핵심 키워드 4~6회 자연스럽게 반복

3. 사진 + 동영상 포함 여부 (20%)
   → 실제 출장 사진 필수 (알트태그에 키워드 포함)

4. 포스팅 체류시간 (15%)
   → 증상/원인/해결 구조로 독자가 끝까지 읽도록 유도

5. 외부 링크 및 태그 (15%)
   → area 페이지 링크 + 지역명 태그 필수
```

### 7-2. AI 검색(CUE:) 최적화 포인트

```
1. 구체적 질문+답변 형식 (FAQ)
   → "Q: 에어컨 성에가 왜 낄까요? A: 냉매 부족이 주원인..."
   
2. 지역+서비스 명시적 언급
   → "안양시 에어컨수리 출장 서비스" 명확히 기재

3. 공식 사이트 링크 연결
   → 신뢰도 신호로 링크 필수

4. E-A-T 강화
   → 실제 출장 사진 + 작업 경험 서술 = 전문성 증명
```
